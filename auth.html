Author Guidelines

Welcome to AMRP 2023. In order to submit an extended abstract through this website, you would need to follow a simple 3-step procedure listed below.

Step 1:Get Register- To register and create an author profile. This allots you a unique author page which helps the website to recognize you. And it is Free! Please skip this step if you are already a register. Your e-mail ID, provided by you in the registration form is considered as registered e-mail ID by us, and the same would be used by us for further communication with you. To begin with, your e-mail ID and Password would be mailed to you. Please mail us, using your registered mail ID, at: support@amrp2023.in in case you don't receive this e-mail.

Step 2: Log in- In order to submit a extended abstract you need to log in to the web account using your E-mail ID and Password.

Step 3: Submit an abstract- You can now submit your extended abstract paper using the Submit Abstract link. Please note that all your co-authors must be members of this website and while submitting the paper you would need to provide their working email id and mobile no. When you submit an abstract, a Unique abstract ID is allotted to it. Please use this ID for any queries regarding the abstract, in future.

Please use the Microsoft Word Templates available at the DOWNLOAD Section of conference website to prepare manuscripts and abstracts. The Microsoft word (DOC) files of abstract has to be uploaded at the website.
